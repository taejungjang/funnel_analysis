# 이커머스 로그데이터를 통한 퍼널 분석

본 프로젝트는 **Google Merchandise Store**의 로그데이터를 활용하여 사용자의 구매 여정을 분석하고, 이탈이 발생하는 지점을 진단하여 전환율 개선을 위한 액션 플랜을 도출한 프로젝트입니다.

---

## 1. 분석 배경 및 목적
* **퍼널 분석 필요성**: 단순히 최종 구매율만 확인하면 중간 과정의 이탈 원인을 파악하기 어렵습니다.
* **데이터 기반 진단**: 방문부터 구매까지의 단계를 세분화하여 UX 개선 및 마케팅 최적화 지점을 찾고자 합니다.

---

## 2. 데이터 개요
* **데이터셋**: `ga4_obfuscated_sample_ecommerce` (GA4 샘플 데이터)
* **기간**: 2020/11/01 – 2021/01/31 (3개월)
* **핵심 지표**: `event_name`(이벤트 로그), `user_pseudo_id`(고객 고유 식별자)

---

## 3. 퍼널 정의 및 전처리
### 분석 단계 (6단계)
1. **Page View**: 웹사이트 방문
2. **View Item**: 상품 상세 페이지 조회
3. **Add to Cart**: 장바구니 담기
4. **Begin Checkout**: 결제 시작 (주소 입력 등)
5. **Add Payment Info**: 결제 정보 입력
6. **Purchase**: 최종 구매 완료

### 데이터 전처리
* **사용자 기준 집계**: 중복을 제거한 고유 사용자 수 기반 분석
* **로그 정합성 보정**: 상위 단계 로그가 누락되었더라도 하위 단계 기록이 존재할 경우 상위 단계에 포함하여 누적 수치 보정

---

## 4. 분석 결과 및 인사이트

### 단계별 전환율 (Conversion Rate)
| 분석 단계 | 사용자 수 | 단계별 전환율 |
| :--- | :---: | :---: |
| Page View (방문) | 269,821 | - |
| View Item (상품 조회) | 61,335 | **22.7%** |
| Add to Cart (장바구니) | 16,603 | 27.0% |
| Begin Checkout (주소입력) | 9,717 | 58.5% |
| Add Payment Info (결제정보) | 5,752 | 59.1% |
| Purchase (최종 구매) | 4,419 | 76.8% |

* **전체 전환율**: 방문 대비 최종 구매율은 **1.6%**로 업계 평균(2.3%) 대비 낮은 수준입니다.
* **주요 병목 지점**: 상품 조회 도달률이 **22.7%**로 업계 기대치(40%)에 미치지 못하는 심각한 병목 현상이 발견되었습니다.

---

## 5. 가설 검정: 유입 경로별 효율 분석
* **가설**: 유입 경로(Referral, Organic, CPC)에 따라 상품 조회 전환율에 차이가 있을 것이다.
* **검정 결과 (카이제곱 검정)**: p-value < 0.05로 통계적으로 유의미한 차이가 확인되었습니다.
    * **Referral**: 24.8%
    * **Organic**: 22.0%
    * **CPC (유료광고)**: 20.0%
* **결론**: 유료 광고(CPC)를 통한 유입이 가장 낮은 전환율을 보이고 있어 광고 타겟팅 및 랜딩 페이지 개선이 시급합니다.

---

## 6. 액션 플랜 (Action Plan)
1. **랜딩 페이지 최적화**: 광고 클릭 시 메인 페이지가 아닌 관련 상품의 상세 페이지로 직접 연결하여 이탈률 감소.
2. **개인화 추천 시스템**: 메인 페이지 및 카테고리 페이지 내 추천 알고리즘을 강화하여 상품 조회 단계로의 유도 증대.
3. **제품 시장성(PMF) 재점검**: 낮은 도달률은 상품 자체의 매력도 문제일 수 있으므로, 타겟 고객층에 맞는 상품 라인업 개선 검토.
